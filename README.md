# Criando aúdios em diferenstes idiomas através da IA ;)

## 📒 Descrição
Criando áudios a partir de texto em idiomas diferentes usando IA Generativa.

## 🤖 Tecnologias Utilizadas
https://elevenlabs.io/

## 🧐 Processo de Criação
Utilizando o mesmo texto traduzido para idiomas diferentes, foi gerado o aúdio em idiomas diferentes (português, Inglês e Alemão)

## 🚀 Resultados
Em segundos é possível criar aúdio em diversos idiomas diferentes. Essa ferramenta pode ser útil no aprendizado e um novo idioma ou apresentar conteúdo para públicos diferentes.
O aúdio fica igual à alguém gravando em um estúdio profissional.
